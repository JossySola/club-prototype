Responsive Club Website

About

The page linked to this style page is FICTIONAL.


This is part of the project from Codecademy called: Responsive Club Website, and the aim is to deploy a website about a club primarly using responsive measures such as EM, REM and percentage units.


The project has to use images which need to resize accordingly to the screen size, also it needs to use media queries to successfuly modify the content according to the screen size.


I thought about a club of people who put their physical and mental health as a priority. In addition, each one of them would help other members depending on their knowledge about certain subjects, them being: sports, athletism, therapy, meditation, etc.


The layout of the website is thought to be a representation of the day cycle and the beautiful color palette we see in the sky as we work through our activities.
